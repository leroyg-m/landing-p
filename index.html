Coding Projects

This project is a Dungeons and Dragons-inspired role-playing word game developed in Java. It showcases the use of inheritance and data structures to create dynamic characters, each equipped with specific tools and weapons for attack and defense. Players take turns until there is a winner or a lone survivor, adding an element of strategy and competition. Collaborating with Professor Diane Mueller, I focused on implementing core object-oriented principles, enhancing both the gameplay experience and my understanding of Java's capabilities.
